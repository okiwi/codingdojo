# Kata FizzBuzz F#

Résultat du Kata "FizzBuzz" d'initiation au Coding Dojo en F# durant le Geek Camp 29/09/2019. 

## Bien démarrer

### Prérequis

* dotnet core installé

### Lancer les tests

#### Ligne de commande

`dotnet test`

#### Rider

Menu `Tests > Run Unit Tests`

## Outils utilisés

* FSharp Core
* Microsoft.NET.Test.Sdk
* NUnit
