# Kata FizzBuzz ES6

Résultat du Kata "FizzBuzz" d'initiation au Coding Dojo durant le Geek Camp 26/09/2019. 

## Bien démarrer

### Prérequis

* nodejs / npm installés
* notions minimum de Javascript (idéalement ES >= 6)

### Installation

`npm install`

## Lancer les tests

`npm test`

## Outils utilisés

* Mocha
* Chai
* Babel
